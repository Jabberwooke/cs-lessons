Lists

An ordered, changeable collection that can hold any mix of values.

1
What is a List?

A list is like a numbered shelf where each slot holds one item. You can add things, remove things, and change what's in any slot. Lists are one of Python's most-used tools.

List: fruits = ["apple", "banana", "cherry", "date"]
"apple"index 0
"banana"index 1
"cherry"index 2
"date"index 3
Analogy

Think of a list like a grocery list on paper — you can cross things off, add new items, and swap things around as much as you want.

2
Creating and Accessing Lists
python
# Create a list
fruits = ["apple", "banana", "cherry"]

# Access by index (starts at 0)
print(fruits[0])       # apple
print(fruits[2])       # cherry
print(fruits[-1])      # cherry  (last item)

# Slicing — get a portion
print(fruits[0:2])     # ['apple', 'banana']

# Lists can hold mixed types
mixed = ["Alice", 17, True, 3.14]
3
Modifying Lists
python
fruits = ["apple", "banana", "cherry"]

# Change an item
fruits[1] = "blueberry"

# Add to the end
fruits.append("date")

# Insert at a specific position
fruits.insert(1, "avocado")

# Remove an item by value
fruits.remove("cherry")

# Remove by index
fruits.pop(0)

# Sort the list
fruits.sort()

print(fruits)
4
Looping Through a List
python
scores = [88, 92, 75, 100, 67]

# Loop through items
for score in scores:
    print(f"Score: {score}")

# Loop with index using enumerate()
for i, score in enumerate(scores):
    print(f"Student {i+1}: {score}")

# Useful list functions
print(len(scores))        # number of items: 5
print(sum(scores))        # total: 422
print(max(scores))        # highest: 100
print(min(scores))        # lowest: 67

Tuples

An ordered, unchangeable collection — like a locked list.

1
What is a Tuple?

A tuple is like a list, but once you create it you can't change it. You can't add, remove, or swap items. This makes tuples perfect for data that should stay the same — like GPS coordinates or RGB colors.

Analogy

A tuple is like a printed receipt — it records exactly what happened and you can read it any time, but you can't go back and change the items on it.

2
Creating and Using Tuples
python
# Create a tuple with parentheses
coordinates = (34.7465, -92.2896)
rgb_red      = (255, 0, 0)
student      = ("Alice", 17, "11th grade")

# Access by index — same as lists
print(coordinates[0])    # 34.7465  (latitude)
print(student[0])         # Alice

# Tuple unpacking — assign each value to a variable
lat, lon = coordinates
print(f"Lat: {lat}, Lon: {lon}")

name, age, grade = student
print(f"{name} is in {grade}")

# You CAN loop through a tuple
for value in rgb_red:
    print(value)
3
Tuples are Immutable
python
point = (10, 20)

# This will cause an ERROR:
# point[0] = 99   ← TypeError: 'tuple' object does not support item assignment

# But you CAN reassign the whole variable
point = (99, 20)   # this works — creating a brand new tuple

# Convert between list and tuple
my_list  = [1, 2, 3]
my_tuple = tuple(my_list)   # list → tuple
back_to_list = list(my_tuple) # tuple → list

Libraries

Pre-built collections of code you can import and use in your programs.

1
What is a Library?

A library (also called a module) is a file full of pre-written Python functions and tools. Instead of writing code from scratch, you import a library and use what's already built.

Analogy

A library is like a toolbox. You don't build a hammer from raw materials every time you need one — you just open the toolbox and grab it. Python's libraries are pre-made tools for math, dates, files, web requests, and much more.

2
Importing Libraries
python
# Import the whole library
import math
print(math.sqrt(144))     # 12.0
print(math.pi)              # 3.141592...

# Import just what you need
from math import sqrt, pi
print(sqrt(64))             # 8.0  (no "math." prefix needed)

# Import with a short alias (nickname)
import random as r
print(r.randint(1, 10))    # random number 1–10
3
The math Library
python
import math

print(math.sqrt(25))       # 5.0  — square root
print(math.floor(4.9))     # 4    — round down
print(math.ceil(4.1))      # 5    — round up
print(math.pow(2, 8))       # 256.0 — 2 to the 8th power
print(math.log(100, 10))   # 2.0  — log base 10
print(math.pi)              # 3.14159...
print(math.factorial(5))   # 120  — 5!
4

Stacks

Last In, First Out — the last item added is the first one removed.

1
What is a Stack?

A stack is a data structure where items are added and removed from the same end — the top. The last item you pushed on is the first one you pop off. This is called LIFO: Last In, First Out.

Analogy

Think of a stack of lunch trays in a cafeteria. You always take the tray from the top (the last one placed there), and when trays are cleaned, they go back on top.

Stack — items added/removed from the TOP
"email_draft" ← TOP (most recent) ← pop() / push()
"photo_edit"
"spreadsheet"
"browser" BOTTOM (oldest)
2
Implementing a Stack with a List

Python's built-in list works perfectly as a stack. Use .append() to push and .pop() to remove from the top.

python
# A stack is just a list used in a specific way
undo_stack = []

# PUSH — add to the top
undo_stack.append("typed 'Hello'")
undo_stack.append("typed ' World'")
undo_stack.append("deleted 3 chars")

print(undo_stack)
# ['typed Hello', 'typed  World', 'deleted 3 chars']

# POP — remove from the top (undo last action)
last_action = undo_stack.pop()
print(f"Undoing: {last_action}")
# Undoing: deleted 3 chars

# PEEK — see top without removing
top = undo_stack[-1]
print(f"Next to undo: {top}")

# Check if empty
if not undo_stack:
    print("Nothing to undo!")
3
Real-World Uses of Stacks
  • Undo/Redo in text editors and design apps
  • Browser back button — each page visited is pushed; hitting back pops it
  • Function call stack — Python tracks which function called which using a stack
  • Parenthesis matching — compilers use stacks to check if brackets are balanced
python — real example: check balanced brackets
def is_balanced(expression):
    stack = []
    pairs = {')': '(', ']': '[', '}': '{'}

    for char in expression:
        if char in '([{':
            stack.append(char)
        elif char in ')]}':
            if not stack or stack.pop() != pairs[char]:
                return False

    return len(stack) == 0

print(is_balanced("(a + b) * [c - d]"))   # True
print(is_balanced("(a + [b)"))             # False

Queues

First In, First Out — the first item added is the first one removed.

1
What is a Queue?

A queue is a data structure where items are added at the back and removed from the front — just like a real-world line. This is called FIFO: First In, First Out.

Analogy

A queue is exactly like a line at the school lunch counter. The first student in line gets served first. New students join at the back and wait their turn.

Queue — items added at BACK, removed from FRONT
FRONT →
"Alice" ← dequeue
→
"Bob"
→
"Carlos"
→
"Diana" enqueue →
← BACK
2
Using collections.deque for Queues

You could use a list, but Python's collections.deque is faster and purpose-built for queues.

python
from collections import deque

# Create an empty queue
lunch_line = deque()

# ENQUEUE — add to the back
lunch_line.append("Alice")
lunch_line.append("Bob")
lunch_line.append("Carlos")

print(lunch_line)
# deque(['Alice', 'Bob', 'Carlos'])

# DEQUEUE — remove from the front
served = lunch_line.popleft()
print(f"Now serving: {served}")
# Now serving: Alice

# PEEK at the front without removing
next_up = lunch_line[0]
print(f"Next in line: {next_up}")
# Next in line: Bob

# Check how many are waiting
print(f"People in line: {len(lunch_line)}")
# People in line: 2
3
Simulating a Queue — Full Example
python — printer queue simulation
from collections import deque

printer_queue = deque()

# Students send jobs to the printer
printer_queue.append("Math homework - Alice")
printer_queue.append("Essay draft - Bob")
printer_queue.append("Science lab - Carlos")

print(f"Jobs in queue: {len(printer_queue)}")

# Process each job in order
while printer_queue:
    job = printer_queue.popleft()
    print(f"Printing: {job}")

print("All jobs complete!")
